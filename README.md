# OSASI_API

---

## How to Setup

1.  Clone Repository
    ```
      git clone https://github.com/ketdev-og/OSASI_API.git
    ```
    
2. Navigate to Directory
     ```
      cd OSASI_API
    ```
3. Run 
     ```
      mvnw clean install
     ```
4. Start springboot app 
     ```
      mvnw spring-boot:run
     ```

navigate to "http://localhost:9000/swagger-ui/index.html" for api documentation
